# howtocode

source code for my blog made with racket/pollen!


## instructions 
Reset cache(required every racket update):

```raco pollen reset```

Distribute:

while in src

```raco pollen render```

```raco pollen publish```

## editing:
if using vscode, type in `lz` for the lozenge character which will popup a snippet dropdown
