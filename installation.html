Installation

If these are not already on, please turn them on, otherwise you don’t need to do anything.

  1. Download Racket https://download.racket-lang.org/
  2. Edit->Preferences->Editing->General Editing->Show line numbers
  3. View->Use Horizontal Layout (optional but more space)
  4. In the Bottom Left Corner, Choose Language -> Beginning Student

Optional:

  1. For autocompletion: File->Package Manager->drcomplete
  2. Edit->Preferences->Editing->General Editing->Enable automatic parentheses, square brackets, and quotes (it can lead to editing problems if you’re not paying attention)
